# Drawn Together

# Pre-requisites

- Node.js v21 (nvm optional)
- npm

# Installation

1. Clone repository `git clone https://github.com/icedtrees/drawn-together`
2. `npm install`

# Running locally

Development (frontend + backend with watch):

1. Run `npm run dev`
   1. This will run the backend in development mode with hot reload
   2. The frontend bundle is rebuilt on change
2. Access at http://localhost:3000/

Production build:

1. Run `npm run build`
2. Run `npm start`
3. Access at http://localhost:3000/
